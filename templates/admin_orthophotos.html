About Orthophotos

Orthophotos are georeferenced images displayed as map overlays. This system supports Cloud Optimized GeoTIFFs (COG) stored in Dropbox.

  • Name: Unique identifier for the orthophoto
  • Dropbox Path: Path to the COG file in Dropbox
  • Status: Active orthophotos are displayed on the map. Only one should be active at a time.
  • Bounding Box: Optional geographic extent (xmin, ymin, xmax, ymax) for map fitting

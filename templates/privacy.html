Privacy Policy

Last Updated: November 2025

Overview

The Soqotra Rock Art Database is a private research application used for managing archaeological documentation. This application accesses Dropbox only to store and retrieve files belonging to the application owner.

Data Collection

This application does not collect, store, or share any personal data from users other than the application administrator. The Dropbox integration is used solely to access files in the administrator's Dropbox account.

Dropbox Access

The application uses Dropbox API to:

  • Read image files for archaeological documentation
  • Generate temporary links for displaying images
  • Access geospatial data (GeoTIFF files)

All data accessed belongs to the application owner and is not shared with third parties.

Security

Access to this application requires authentication. All data transmission is encrypted using HTTPS.

Contact

For questions about this privacy policy, please contact the application administrator.

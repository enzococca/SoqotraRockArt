How to Use
Creating a Database:

Click "Create New Database" to create an empty database with the same schema as the current one.

Switching Databases:

Currently requires manual configuration. Update the DATABASE_URL environment variable in your .env file:

DATABASE_URL=sqlite:///your_database.db

Then restart the Flask application.

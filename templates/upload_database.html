Upload Existing Database

Upload an existing SQLite database

Upload a .db file that was created previously. The database should have the same schema as this application.

Only .db (SQLite) files are accepted
If checked, the current database will be backed up and this database will become active. You will need to refresh the page and login with credentials from the new database.
Important:
  • The database must be a valid SQLite file
  • It should contain compatible tables (rockart, images, users, type_descriptions)
  • Image paths should be relative or you may need to re-upload images
  • If a database with the same name exists, rename it first

After Uploading

To use the uploaded database:

  1. The database will appear in the "Manage Databases" list
  2. To switch to it, update your .env file: 
    DATABASE_URL=sqlite:///your_uploaded_database.db
  3. Restart the Flask application
  4. Login with the credentials from that database

Note: You can also:

  • Export the current database data to Excel
  • Merge data from multiple databases manually
  • Keep multiple databases for different projects/sites

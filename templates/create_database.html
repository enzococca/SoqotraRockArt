Create New Database

What does this do?

This will create a new empty SQLite database with the same schema as the current database. The new database will include:

  • Users table (for authentication)
  • Rock art records table
  • Images table
  • Type descriptions table
Use a descriptive name. The .db extension will be added automatically if not provided.
Naming Suggestions:
  • soqotra_site1.db - Database for a specific site
  • fieldwork_2024.db - Database for a specific campaign
  • backup_20240120.db - Backup database with date
  • test_database.db - Testing/development database
Note: The new database will be created empty. You'll need to:
  • Create a new admin user manually (or register through the app)
  • Import type descriptions if needed
  • Add records and images
Cancel
After Creating the Database

To start using the new database:

  1. Update your .env file with the new database path: 
    DATABASE_URL=sqlite:///your_new_database.db
  2. Restart the Flask application
  3. Register a new admin user or create one using Flask shell: 
    flask shell
>>> from models import User, db
>>> user = User(username='admin', email='admin@example.com')
>>> user.set_password('your_password')
>>> db.session.add(user)
>>> db.session.commit()
  4. Start adding records!
